# basisverzekering-stats
This repository contains the statistical analysis of the averaged cost per person with basic health insurance from different income groups, during the period of 2009 - 2013.

This project aimed to answer to following question: When assessing the averaged cost per person with basic health insurance in The Netherlands, are there significant differences between the income groups as well as across the years ? This information can help authorities to investigate which part(s) of the society has/have been more impacted with basic insurance costs, and therefore define health policies/actions that can benefit this/these group(s). 

The dataset used in this project is publicly available at the Centraal Bureau voor de Statistiek (CBS) here: https://opendata.cbs.nl/#/CBS/nl/dataset/81827NED/table?ts=1774895242480
